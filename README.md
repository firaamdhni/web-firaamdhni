# 🌐 Web Personal - Fira Ramdhani

Website personal sederhana yang berisi informasi tentang profil, pengalaman, dan kontak saya.

## 📌 Deskripsi

Project ini dibuat untuk menampilkan identitas diri secara online dalam bentuk website statis menggunakan HTML dan CSS.

Website ini terdiri dari:

* Halaman Beranda (Home)
* Halaman Tentang Saya (About)
* Styling menggunakan CSS

## 🛠️ Teknologi yang Digunakan

* HTML5
* CSS3
* Git
* GitHub

## 📂 Struktur Project

```
web-personal/
│── index.html
│── about.html
│── style.css
│── README.md
```

## 🚀 Cara Menjalankan Project

1. Download atau clone repository ini
2. Buka folder project
3. Klik dua kali file `index.html`

Atau bisa menggunakan Live Server di VS Code.
